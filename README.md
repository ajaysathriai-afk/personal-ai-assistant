# 🤖 Personal AI Assistant

An intelligent AI-powered chat assistant with memory and multiple conversation modes.

## ✨ Features

- 💬 Natural conversations with context memory
- 🎭 Multiple modes (General, Tutor, Career Coach, Deep Thinker)
- 🎨 Beautiful neon-themed UI with smooth animations
- 🧠 Powered by LangChain + OpenAI GPT-4
- ⚡ Real-time responses with typing indicators

## 🛠️ Tech Stack

- **LangChain**: Conversation management and memory
- **OpenAI GPT-4**: Language model
- **Streamlit**: Web interface
- **Python 3.x**: Backend

## 📦 Installation
```bash
# Clone repository
git clone https://github.com/YOUR_USERNAME/personal-ai-assistant.git
cd personal-ai-assistant

# Create virtual environment
python3 -m venv venv
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Create .env file
echo "OPENAI_API_KEY=your_key_here" > .env

# Run application
streamlit run app.py
```
